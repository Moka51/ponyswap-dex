# PonySwap Exchange


[PonySwap](https://ponyswap.com/) is an automated market maker (“**AMM**”) that allows two tokens to be exchanged on the [Binance Smart Chain](https://www.binance.org/en/smartChain) (BSC). It is fast, cheap, and allows anyone to participate.

This repo is responsible for the **exchange** interface of the AMM: [dex.ponyswap.com](https://dex.ponyswap.com/)

If you want to contribute, please refer to the [contributing guidelines](./CONTRIBUTING.md) of this project.
If you want to list a token, refers to the [listing guidelines](./listing.md).
